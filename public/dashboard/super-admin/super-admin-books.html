Import Books from CSV or Excel

info File Format Requirements (CSV or Excel)
Required columns (in order):
title,author,isbn,category,quantity
Validation Rules:
  • Required fields: title, author, isbn, category (rows with missing fields will be skipped)
  • Quantity: Defaults to 1 if empty; use 0 for borrowed/missing books
  • Duplicate ISBNs: Rows with ISBNs already in the database will be skipped
  • Status rules: Quantity ≥ 1 = "available"; Quantity = 0 = "borrowed" or "missing" based on borrowing records
